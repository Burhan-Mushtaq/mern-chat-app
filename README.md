# 💬 MERN Real-Time Chat App

A full-stack real-time chat application built using the MERN stack and Socket.io. This project features private messaging using rooms and a modern WhatsApp-like UI.

---

## 🚀 Features

- 🔐 Real-time messaging with Socket.io
- 👥 Private chat using rooms
- ⚡ Instant message delivery
- 💬 WhatsApp-inspired UI
- 🔄 Live connection handling
- 🧠 Event-driven architecture

---

## 🛠️ Tech Stack

### Frontend:
- React (Vite)
- Tailwind CSS
- Socket.io-client

### Backend:
- Node.js
- Express.js
- Socket.io
